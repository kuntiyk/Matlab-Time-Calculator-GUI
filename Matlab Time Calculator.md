# Matlab Time Calculator GUI

A simple MATLAB GUI application developed to manage time calculations efficiently.

## Features
- *Add Time:* Input minutes and seconds to add to the total duration.
- *Subtract Time:* Deduct specific minutes/seconds from the total.
- *Reset:* Clears all calculations and resets the counter to zero.

## How It Works
The app uses a user-friendly interface with:
- 3 Control Buttons (Add, Subtract, Reset)
- 2 Edit Fields for user input
- A dynamic Label to display the result

## Technologies Used
- MATLAB
- MATLAB App Designer / GUI
